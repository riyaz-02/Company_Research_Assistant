# Company Research Assistant

A Laravel-based application for company research and analysis.

## Project Information

- **Project Name**: Company Research Assistant
- **Author**: SkRiyaz
- **Company**: EightFold AI
- **Purpose**: Recruitment Assignment

## Requirements

- PHP >= 8.2
- Composer
- Node.js and NPM

## Installation

1. Clone the repository
2. Install PHP dependencies:
   ```bash
   composer install
   ```
3. Copy environment file:
   ```bash
   cp .env.example .env
   ```
4. Generate application key:
   ```bash
   php artisan key:generate
   ```
5. Install Node dependencies:
   ```bash
   npm install
   ```

## Development

Run the development server:
```bash
php artisan serve
```

## Framework

This project is built using [Laravel](https://laravel.com) framework.
